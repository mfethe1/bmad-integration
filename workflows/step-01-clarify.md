# Step 1: Clarify

**Goal:** Ensure complete understanding of the user request and establish clear constraints before moving to planning.

## Instructions
1. Review the user's initial request.
2. Identify missing information, undefined constraints, and potential edge cases.
3. If ambiguities exist, list exactly 3 specific questions for the user.
4. Do not begin planning or implementation.

## Checkpoint (HALT)
**STOP AND WAIT FOR USER INPUT**. Only proceed to `step-02-plan.md` once the user has answered the clarifying questions.
